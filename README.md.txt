## EBAC Shoes Project, Responsive Site with Grid Display

This project is part of the Front-End Engineer course at EBAC, module 11. Our main goal is to develop a responsive site that works seamlessly on a variety of devices, including tablets, smartphones, and desktops, leveraging the power of the Grid and other responsiveness tools.

### Key Project Objectives:

- Create an attractive and efficient layout using the Grid.
- Ensure the site is easily navigable on different screen sizes.
- Enhance the user experience on mobile and desktop devices.
- Implement modern web development practices.

Feel free to explore our project and contribute! Your contribution is welcome. Together, we are learning to create high-quality and adaptable web experiences.

### Technologies Used:

- HTML
- CSS

Este projeto faz parte do curso de Engenheiro Front End na EBAC, módulo 11. Nosso objetivo principal é desenvolver um site responsivo que funcione perfeitamente em uma variedade de dispositivos, incluindo tablets, smartphones e desktops, aproveitando o poder do Grid e outras ferramentas de responsividade.
